# Power BI Sales Analysis

This repository contains a Power BI dashboard focused on analyzing sales,
profitability, customer performance, and delivery efficiency.

## Project Scope
- Sales and profit analysis
- Customer performance analysis
- Delivery time and shipping mode analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Dashboard Design

## Dashboard Structure
- Overview page: Key sales, profit, loss, and margin metrics
- Customer Analysis page: Customer sales performance and trends
- Delivery Status page: Delivery time and shipment analysis

## Files
- Screenshots: Dashboard screenshots
- Video: Dashboard screen recording

This project demonstrates practical Power BI dashboard development,
including data preparation, KPI creation, and visual analysis.
